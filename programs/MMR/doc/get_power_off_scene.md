
得到关机情景的接口文档
========================

### API编号

### 功能简介
* 对数据进行处理，判断是否属于关机情景，如果是则通过http请求把数据传给ptop

### 参数格式

### 传入参数

 参数             |参数说明         |  类型       |   示例         |是否允许为空| 
------------------|-----------------|-------------|----------------|------------|
 accountID        | 用户帐号编号    | string      |  aaaaaaaaaa    |否          | 
 tokenCode        | 经度  	    | string      |  xxxx	   |否          | 
 IMEI 	          | 纬度	    | string      |  xxxxxxxx	   |否          | 
 model	          | 集合	    | string      |  SG900	   |否          | 
 poweroff         | 关机特有字段    | string      |  true	   |否          |
 

### 返回结果参数

参数                | 参数说明
--------------------|-------------------------------------------
无                  | 无
